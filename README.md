# Always Coding — static page

Simple static page that shows a message when the button is clicked.

Modernized and includes a link to the TikTok handle: @code_it_your_way

How to host on GitHub Pages:

1. Create a repo and push the `always-coding` folder contents to the repo root.
2. In GitHub go to Settings → Pages and select the `main` branch / root folder.
3. Visit the provided GitHub Pages URL.

Tips:
- Replace `me.jpeg` with your profile image (recommended size 300x300px).
- The message text copies to clipboard when you click it after revealing.
